void vfo_base::set_gain_mode ( gain_state  state)

Set VFO gain mode (H or L).

Parameters
statevfo_base::gain_state::low or vfo_base::gain_state::high. The low gain setting will be used for normal data reading and switched to the high gain setting during SYNC field reading. mfm_codec class will control the gain setting through this function.
